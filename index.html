클라우드 플랫폼

MICROSOFT AZURE
Microsoft Azure는 Microsoft의 클라우드 컴퓨팅 플랫폼으로, 분석, 컴퓨팅, 데이터베이스, 모바일, 네트워킹, 저장소 및 웹 서비스가 통합된, 점점 더 성장하고 있는 서비스 컬렉션입니다. 이를 통해, 작동이 더 빨라지고, 더 많은 성과를 거두며, 비용을 절감할 수 있습니다.

GOOGLE CLOUD PLATFORM
구글 클라우드 플랫폼은 실제 구글 서비스를 위해서 사용되고 있는 컴퓨팅 기술을 활용하는 제품으로써 컴퓨팅, 스토리지, 빅 데이터 등등의 서비스를 제공합니다. 구글 클라우드 플랫폼을 통해 막대한 비용을 지출하지 않고도 직접 인프라를 관리하고 구글이 구축해 놓은 인프라 위에서 구글과 동일한 규모와 속도로 앱을 만들고 운영할 수 있다는 장점이 있습니다. 이로 인해 개발자는 서버를 관리하는 시간을 줄이고, 서비스 개발과 혁신적인 사업을 구축하는 데 집중할 수 있습니다.

AMAZON AWS
Amazon Web Services에서는 조직이 더 빠르게 발전하고, IT 비용을 절감하고, 애플리케이션을 조정하도록 지원하는 글로벌 컴퓨팅, 스토리지, 데이터베이스, 분석, 애플리케이션 및 배포 서비스를 제공하고 있습니다.



자세한 사항은 영업팀(sales@bncsoft.com)으로 문의주십시오.

컨택센터 클라우드 서비스

컨택센터 토털 아웃소싱 / ASP서비스
검증된 글로벌 스탠다드 컨택센터 솔루션을 기반으로 초기투자 비용없이 가장 저렴하고 안정된 컨택센터 서비스로 귀사의 비즈니스를 지원합니다.
  - 컨택센터 토털 아웃소싱: 200석 규모의 콜센터 상면, 검증된 컨택센터 인프라, 숙련된 상담사 지원
  - 컨택센터 ASP: 고객사 지정 위치에서 컨택센터 솔루션 임대 사용

기업 컨택센터를 위한 Noogoo 서비스
기업 컨택센터 시스템과 연동하여 고객센터와 음성통화를 하는 고객에게 모바일 컨텐츠를 함께 제공하는 서비스입니다.
  - Info Push: 통화 연결 중 배너 표출, 통화 종료 후 모바일 페이지 화면 전송
  - Visual Voice: ARS 시나리오 메뉴를 텍스트로 표출
  - VDL/Agent Push: 통화 중/종료 후 고객 요청 화면 전송

개인 마케팅을 위한 Popconi 서비스
고객이 사업자 스마트폰 또는 유선 전화로 걸면 통화 연결중, 통화 종료 후 사업자의 홍보 마케팅 컨텐츠를 자동 팝업해 주는 서비스입니다.
  - 통화연결 중: 기업 배너 정보 표출
  - 통화종료 후: 모바일 페이지 연결 또는 MMS 메시지 전송
